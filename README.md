[ğŸ–§ SleekXMPP](https://github.com/fritzy/sleekxmpp) (deprecated) was a Python 2.x XMPP protocol library that was heavily used the power industry, used on game consoles, used for the examples in [XMPP: The Definitive Guide published by O'Reilly](https://www.oreilly.com/library/view/xmpp-the-definitive/9780596157524/). [ğŸµ Dulcimer](https://github.com/fritzy/Dulcimer) and it's dependency [VeryModel](https://github.com/fritzy/VeryModel) was an ORM wrapper for leveldb. It gained some popularity and we actively used it at [&yet](https://andyet.com) for bots and small projects.
